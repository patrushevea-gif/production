# Санкционный чекер

Новый автономный мини‑проект для юридической проверки контрагентов по открытым санкционным источникам.

## Что делает

- Принимает ИНН или название компании.
- Генерирует набор прямых ссылок на международные и российские открытые источники.
- Ускоряет ручную проверку без изменения существующих проектов в репозитории.

## Запуск локально

```bash
cd sanctions-checker
python3 -m http.server 4173
```

После запуска откройте: <http://localhost:4173>

## Примечание

Инструмент не хранит данные и не делает автоматических юридических выводов — он помогает быстро открыть релевантные реестры и официальные страницы для ручной проверки.
